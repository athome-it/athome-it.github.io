Mullvad Leta mit voreingestellter Sprache und Engine nutzen

Leta von Mullvad unterstützt zwar Sprach- und Länderparameter, diese werden jedoch aktuell nicht dauerhaft gespeichert. Ein Userscript kann Abhilfe schaffen.

Userscript: Leta automatisch auf Deutsch und gewünschter Engine

// ==UserScript==
// @name         Leta (Mullvad) Sprache/Region Auto-Setter
// @match        https://leta.mullvad.net/*
// @grant        none
// ==/UserScript==
(function() {
    const url = new URL(window.location.href);
    if (!url.searchParams.get("language")) {
        url.searchParams.set("language", "de");
        url.searchParams.set("country", "de");
        url.searchParams.set("engine", "brave");
        window.location.href = url.toString();
    }
})();

Installation: Das Script erfordert ein Tool wie Tampermonkey (Firefox/Chrome).

Tampermonkey installieren und konfigurieren

  1. Tampermonkey herunterladen:

    • Firefox Add-on
    • Chrome Web Store (Wenn es Chrome noch unbedingt sein muss)

  2. Nach Installation auf das Icon klicken → „Neues Script erstellen“

  3. Obigen Code einfügen, speichern, fertig.

  4. Seite leta.mullvad.net neu laden – Sprache und Engine sollten automatisch gesetzt werden.

Leta-Ergebnisse vergleichen – Brave vs Google

Ein Vergleich der Ergebnisse aus Leta (Brave vs Google) ist maschinell nur schwer automatisierbar, da beide Engines dynamisches HTML erzeugen. Für manuelle Vergleiche eignet sich ein einfaches Webtool.

HTML-Tool zur Ergebnisvergleich (Brave vs Google)

<!DOCTYPE html>
<html lang="de">
<head>
  <meta charset="UTF-8">
  <title>Suchergebnis-Vergleich</title>
  <style>
    body { font-family: sans-serif; display: flex; gap: 2rem; }
    textarea { width: 45%; height: 300px; }
    pre { background: #f0f0f0; padding: 1rem; }
    .diff { color: red; }
  </style>
</head>
<body>
  <textarea id="left" placeholder="Brave-Ergebnisse hier einfügen..."></textarea>
  <textarea id="right" placeholder="Google-Ergebnisse hier einfügen..."></textarea>
  <button onclick="compare()">Vergleichen</button>
  <pre id="output"></pre>

  <script>
    function compare() {
      const a = document.getElementById("left").value.split("\n");
      const b = document.getElementById("right").value.split("\n");
      const result = [];
      const max = Math.max(a.length, b.length);
      for (let i = 0; i < max; i++) {
        if (a[i] !== b[i]) {
          result.push(`🔸 ${a[i] || ''}\n🔹 ${b[i] || ''}\n`);
        }
      }
      document.getElementById("output").textContent = result.join("\n");
    }
  </script>
</body>
</html>

Das Tool funktioniert offline – einfach als .html speichern und lokal im Browser öffnen.

Proxy-Schutz bei Link-Klicks automatisch aktivieren

Viele datenschutzfreundliche Suchmaschinen (z. B. Startpage oder Leta) bieten eine Proxy-Vorschau-Funktion, die beim Klick auf Links genutzt werden kann.

Für Startpage-User empfiehlt es sich, unter Einstellungen die Option “Anonyme Ansicht” als Standard zu setzen. Alternativ kann mit einem Bookmarklet oder Userscript der Proxy automatisch ausgewählt werden (je nach Engine).

URLs mit Parametern als Lesezeichen speichern

Für häufige Suchen mit voreingestellten Engines und Sprache:

  • Leta Brave (deutsch): https://leta.mullvad.net/search?q=%s&engine=brave&country=de&language=de

  • Leta Google (deutsch): https://leta.mullvad.net/search?q=%s&engine=google&country=de&language=de

(Ersetze %s durch deinen Suchbegriff oder nutze die Links in deiner Lesezeichenleiste.)

Browser-Shortcut einrichten (z. B. Firefox/Chrome):

  1. In den Einstellungen unter “Suchmaschinen verwalten” gehen
  2. Neue Suchmaschine hinzufügen: Name „Leta Brave“, Keyword „lb“
  3. Als URL die passende Variante mit %s einfügen
  4. Danach kannst du im Adressfeld z. B. lb Datenschutz eingeben – Suche startet automatisch
